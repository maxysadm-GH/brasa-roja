Phase 2: Visual Branding Implementation

  • Update CSS Color Variables
    Replace existing colors with brand palette throughout the site.
  • Install Brand Fonts
    Add Cinzel (headlines), Lora (body), Parisienne (accents) via Google Fonts.
  • Update Button Styles
    Primary buttons: Roja Red background. Secondary: Gold outline.
  • Add Footer Logo
    Use BrasaRoja_Primary_Light.svg or wordmark in footer with social links.

CSS Variables

:root { /* Primary Brand Colors */ --brasa-gold: #C9882B; --roja-red: #C63333; --deep-ember: #8B2500; /* Neutral Colors */ --charcoal: #2D2D2D; --crema: #F5EFE0; --humo: #8B7355; /* Typography */ --font-display: 'Cinzel', Georgia, serif; --font-body: 'Lora', Georgia, serif; --font-accent: 'Parisienne', cursive; /* Shadows */ --shadow-sm: 0 2px 8px rgba(45, 45, 45, 0.1); --shadow-md: 0 4px 20px rgba(45, 45, 45, 0.15); --shadow-lg: 0 10px 40px rgba(45, 45, 45, 0.2); } /* Dark theme variant */ [data-theme="dark"] { --bg-primary: var(--charcoal); --bg-secondary: #1a1a1a; --text-primary: var(--crema); --text-secondary: rgba(245, 239, 224, 0.7); }

Phase 3: Content & Pages

  • Home Page Hero
    Full-width hero with flame video/image background. "El Arte del Fuego" headline. CTA to reservations.
  • Menu Page
    Digital menu with sections: Starters, From the Grill, Sides, Desserts, Drinks. Include prices.
  • About/Story Page
    "Tradición Familiar" story. Family photos if available. Chef bio.
  • Contact/Location Page
    Address, phone, hours, Google Map embed, reservation form/link.
  • Gallery Page
    Food photography, restaurant interior, kitchen action shots.

Phase 4: Functionality & Integrations

  • Reservation System
    Integrate OpenTable, Resy, or Yelp Reservations. Add prominent "Reserve" button.
  • Online Ordering (Optional)
    For takeout: Square, Toast, or ChowNow integration.
  • Google Analytics
    Set up GA4 tracking. Track reservations, menu views, contact clicks.
  • Google Business Profile Link
    Verify website URL in Google Business. Add structured data for LocalBusiness.
  • Instagram Feed Widget
    Embed live Instagram feed on homepage or dedicated section.

Domain & Hosting Notes

Current URL: brasaroja.lovable.app

Recommended: Purchase brasaroja.com and set up redirect from lovable.app subdomain.
  • Purchase Domain
    Register brasaroja.com via Namecheap, Google Domains, or GoDaddy.
  • SSL Certificate
    Ensure HTTPS is enabled (most hosts include free SSL).
  • Set Up Email
    Create info@brasaroja.com, reservations@brasaroja.com via Google Workspace or Zoho.
