## Brand Elements

### Logo Components
1. **Flame** - Red 3-pointed stylized fire with inner depth
2. **Orbital Swirl** - Gold curved band wrapping the flame base
3. **Rising Embers** - Gold dots trailing diagonally upward
4. **Wordmark** - "BRASA" (gold) + "ROJA" (red)
5. **Tagline** - "Tradición Familiar" in italic script

### Typography
- **Headlines:** Cinzel (serif)
- **Body:** Lora (serif)
- **Script/Tagline:** Parisienne (cursive)
