Setup Instructions

Logo File to Use: Always use Logo_Full/Logo_Transparent.png - the complete logo with "Brasa Roja" and "Tradición Familiar" text. Never use icon-only for signatures.

Step-by-Step Setup

  1. Host Your Logo
    Upload Assets/Logo_Full/Logo_Transparent.png to your web server or image hosting service (Cloudinary, Imgur, or your website's media folder).
  2. Copy the HTML Template
    Choose your preferred template and click "Copy to Clipboard".
  3. Replace Placeholder Values
    Replace [YOUR_HOSTED_URL] with your image URL, and fill in [FULL NAME], [JOB TITLE], etc.
  4. Add to Your Email Client
    Gmail: Settings → See all settings → General → Signature
    Outlook: File → Options → Mail → Signatures
    Apple Mail: Preferences → Signatures

Variables Reference

Variable Description Example
[YOUR_HOSTED_URL] URL where logo is hosted https://brasaroja.com/images
[FULL NAME] Employee's full name Juan Carlos Mendoza
[JOB TITLE] Position title General Manager
[PHONE] Contact phone (555) 123-4567
[EMAIL] Email address juan@brasaroja.com
