Instrucciones de Configuración

Archivo de Logo a Usar: Siempre usá Logo_Full/Logo_Transparent.png - el logo completo con el texto "Brasa Roja" y "Tradición Familiar". Nunca uses solo el ícono para firmas.

Configuración Paso a Paso

  1. Alojar tu Logo
    Subí Assets/Logo_Full/Logo_Transparent.png a tu servidor web o servicio de alojamiento de imágenes (Cloudinary, Imgur, o la carpeta de medios de tu sitio web).
  2. Copiar la Plantilla HTML
    Elegí tu plantilla preferida y hacé clic en "Copiar al Portapapeles".
  3. Reemplazar Valores de Marcador
    Reemplazá [TU_URL_ALOJADA] con tu URL de imagen, y completá [NOMBRE COMPLETO], [CARGO], etc.
  4. Agregar a tu Cliente de Correo
    Gmail: Configuración → Ver toda la configuración → General → Firma
    Outlook: Archivo → Opciones → Correo → Firmas
    Apple Mail: Preferencias → Firmas

Referencia de Variables

Variable Descripción Ejemplo
[TU_URL_ALOJADA] URL donde está alojado el logo https://brasaroja.com/imagenes
[NOMBRE COMPLETO] Nombre completo del empleado Juan Carlos Méndez
[CARGO] Título del puesto Gerente General
[TELÉFONO] Teléfono de contacto (011) 4555-1234
[EMAIL] Dirección de correo electrónico juan@brasaroja.com
