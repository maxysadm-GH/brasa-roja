Brand Launch Checklist

  • Convert logo files to vector formats (AI, EPS, SVG)
  • Order business cards from printer
  • Set up email signatures for all staff
  • Create Instagram/Facebook accounts with branded profile
  • Upload profile photo (icon only, 320x320)
  • Set up Facebook cover image (820x312)
  • Schedule first week of social media posts
  • Print menus (get proof before full run)
  • Set up TV digital menu board
  • Order staff uniforms with embroidered logo
  • Install exterior signage
  • Create Google Business Profile with branding
  • Set up Yelp page with logo and photos
  • Train staff on brand voice and social media guidelines
