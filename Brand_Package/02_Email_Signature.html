Setup Instructions

  1. Host your logo: Upload the Brasa Roja icon to your website or use a service like Imgur, then replace "https://yourdomain.com/images/email-logo.png" with the actual URL
  2. Gmail: Settings → See all settings → General → Signature → Click the image icon to add logo
  3. Outlook: File → Options → Mail → Signatures → New → Paste HTML
  4. Apple Mail: Mail → Preferences → Signatures → Create new → Paste formatted signature
  5. Replace bracketed text [YOUR NAME], [YOUR TITLE], etc. with actual information
  6. Test by sending an email to yourself and checking on mobile
