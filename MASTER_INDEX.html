Quick Start Guide

  1. View brand guidelines: Open BRASA_ROJA_Brand_Guidelines.html in browser
  2. Get logo files: Navigate to Brand_Assets/Logos/ - all SVG files are ready
  3. Convert to PNG/JPG: See README_Asset_Index.md for conversion instructions
  4. Business cards: Open Brand_Package/01_Business_Cards.html, send specs to printer
  5. Set up website: Follow Deliverables/Website_Setup_Checklist.html
  6. Social media: Use Brand_Package/03_Social_Media_Guidelines.html for content strategy
